## 📷 Screenshot Technical Specifications

### Browser Settings
- **Browser:** Chrome or Firefox (latest version)
- **Window Size:** 1920x1080 (Full HD)
- **Zoom Level:** 100% (default)
- **Extensions:** Disable ad blockers and other extensions
- **Developer Tools:** Closed (not visible)

### Screen Capture Settings
- **Format:** PNG (preferred) or high-quality JPG
- **Compression:** Minimal to preserve text clarity
- **File Size:** Under 5MB per image
- **Naming Convention:** `feature-description.png`
- **Folder:** `docs/screenshots/` in project repository

### Quality Standards
- **Text Legibility:** All text must be crisp and readable
- **Color Accuracy:** Gradients and colors display correctly
- **No Artifacts:** Clean capture without compression artifacts
- **Complete Interface:** No cut-off elements or partial content
- **Professional Appearance:** Clean, polished, bug-free interface

## 🎨 Visual Composition Guidelines

### Framing
- **Full Interface:** Show complete page layout with header and footer
- **No Scroll Bars:** Hide browser scroll bars for clean appearance
- **Centered Content:** Ensure main content is centered and balanced

- **Visual Hierarchy:** Important elements clearly emphasized

### Lighting & Colors
- **Consistent Lighting:** Use daylight monitor settings
- **Color Calibration:** Ensure accurate color representation
- **High Contrast:** Maintain readability across all elements
- **Brand Colors:** MediGuide AI blue (#3B82F6) displays accurately
- **Gradient Integrity:** Smooth color transitions without banding

### Content Guidelines
- **Real Data:** Use realistic medical information (not lorem ipsum)
- **Positive Examples:** Show successful interactions and positive outcomes
- **Complete Workflows:** Display end-to-end feature functionality
- **Professional Content:** Appropriate for healthcare context

## 📞 Troubleshooting Common Issues

### Blurry Text
- **Cause:** Incorrect browser zoom or display scaling
- **Solution:** Ensure 100% zoom, check monitor DPI settings

### Color Inconsistency
- **Cause:** Monitor calibration or browser color profile
- **Solution:** Use standard sRGB profile, test on multiple devices

### Partial Content
- **Cause:** Window size too small or browser zoom incorrect
- **Solution:** Set exact 1920x1080 window, use F11 fullscreen mode

### Loading States
- **Cause:** Screenshot taken before content fully loaded
- **Solution:** Wait for all animations and API responses to complete

### Missing Features
- **Cause:** JavaScript errors or API failures
- **Solution:** Check browser console, refresh page, use working test data
